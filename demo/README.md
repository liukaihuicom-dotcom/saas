# Broker SaaS UI Demo

基于shadcn/ui风格的Broker SaaS平台UI演示系统。

## 项目概述
这是一个完整的企业级Broker SaaS平台UI演示，基于实际产品需求文档，按照shadcn/ui设计风格开发。系统覆盖了外汇/CFD Broker的核心经营链路。

## 功能模块

### 1. 系统主页 (index.html)
- 平台概览与快速导航
- 核心业务卡片展示
- 用户角色说明

### 2. 经营总览 (dashboard.html)
- 核心经营指标展示
- 注册转化漏斗分析
- 趋势图表和渠道分布
- 高价值客户列表
- 待处理任务列表

### 3. CRM客户中心 (crm.html)
- 客户列表管理
- 高级筛选和搜索
- 客户状态追踪
- 批量操作功能
- 客户分类标签

### 4. KYC审核中心 (kyc.html)
- 待审核队列管理
- 客户身份资料查看
- 风险评估与检测
- 审核决策功能
- 审核统计报表

### 5. 钱包与支付中心 (wallet.html)
- 出金申请审批
- 入金监控和分析
- 支付通道管理
- 异常支付监控
- 资金统计报表

### 6. 代理中心 (agent.html)
- 代理业绩排行
- 返佣结算管理
- 代理等级体系
- 推广素材统计
- 争议处理功能

## 技术栈
- **前端框架**: 原生HTML/CSS/JavaScript
- **样式框架**: Tailwind CSS
- **图表库**: ApexCharts
- **图标字体**: Font Awesome 6
- **字体**: Google Fonts - Inter

## 设计风格
- 遵循shadcn/ui设计原则
- 现代化的组件设计
- 响应式布局适配
- 状态驱动的UI设计
- 数据可视化展示

## 系统角色
- **管理员**: 全局权限，系统配置
- **销售经理**: 客户管理和跟进制
- **审核专员**: KYC/AML审核
- **财务专员**: 资金管理和出金审批
- **代理运营**: 代理体系管理

## 快速开始
1. 在浏览器中打开 `index.html`
2. 通过左侧导航栏切换不同模块
3. 使用模拟数据进行交互体验
